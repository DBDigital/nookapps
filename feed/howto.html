How to configure your nook to install apps

You'll need to allow your nook to install 3rd party applications to install any of these apps here.

YMMV, but what I've done is to get an adb shell on the nook -- and if you're reading this, you probably have figured out how to do it already! Then, I run (lines broken for readability)

# cd /data/data/
        com.android.providers.settings/databases
# sqlite3 settings.db

Here, you'll find a table called secure, and you need to update this table and set install_non_market_apps to 1. Something like

update secure set value=1
  where name='install_non_market_apps';

ought to do it.
